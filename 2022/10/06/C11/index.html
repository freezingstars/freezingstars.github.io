自定义函数

自定义函数有函数名，返回值类型和函数参数
C 库函数 char *strcpy(char *dest, const char *src) 把 src 所指向的字符串复制到 dest。

char *strcpy(char *strDestination,const char *strSource);
上述中char是函数的返回类型，strcpy是函数名，括号内为函数的参数

函数的参数

实际参数（实参）

传递给函数的数值为实参，比如上文中的swap(a,b)中的ab。此为真实传递给函数的参数，可以使常量、变量、表达式、函数等。在进行函数调用时，必须有实参传递给函数。
表达式示例：2+4
函数示例：get_max(4,7)

形式参数（形参）

实参传递给形参，是函数名括号后的变量。形参只有在调用过程中才实例化。swap(int x,int y)括号内的便是形参，仅做一个形式，在调用函数实参传递前，形参没有意义。在调用完成后被自动摧毁，形参尽在函数内有效。

调用方式有两种：传值调用与传址调用。函数的实参与形参占有不同内存块，对形参的修改不影响实参。传值调用不可影响外部数据，传址可以。

函数内也可简写为return ((x % 400 == 0) || (x % 4 == 0 && x % 100 != 0));
当数值满足表达式时，也可返回一个真值

数组arr传参，实际仅传出第一个元素的地址
